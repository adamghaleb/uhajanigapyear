# Jonny Gap Year

A persuasion site from Adam Fadi to Jonny Quibbuyen. Take the gap year.

## Setup

```bash
npm install && npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion

## Pages

| Route         | What                                  |
| ------------- | ------------------------------------- |
| `/`           | Landing — Jonny's name, thesis, enter |
| `/roundtable` | Jobs × Karpathy × Fadi dialogue       |
| `/sunk-cost`  | The fallacy, visualized               |
| `/the-gap`    | What the degree won't teach           |
| `/permission` | Gap year ≠ dropout                    |
| `/letter`     | Adam's letter to Jonny                |
| `/now`        | Concrete next steps                   |

## Deploy

```bash
npm run build
# or push to Vercel
```
